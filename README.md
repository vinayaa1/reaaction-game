# Reaaction Time Game!
## how the game works:
### 1. choose a theme (dark or cuteness) and then click the start button to begin
### 2. a square will appear on the screen, click it as fast as you can once it turns green
### 3. you will be given 5 rounds to click the square
### 4. after the rounds are complete, your average time (reaction speed) will be shown
### 5. a leaderboard feature is being created (in progress)
## key features:
### - reaction time
### - multiple rounds gameplay system
### - theme toggle
### - sound effects
### - average score calculations
